# Little Lemon Restaurant Website

This project is a React web application created for the Little Lemon restaurant.
The goal of the project is to build a responsive restaurant website with a table booking feature using React, CSS, and basic API logic.
This project was developed as part of a front-end development learning program.

## Features

* Home page with hero section and specials menu
* Menu section with food images and descriptions
* Table booking form
* Select number of guests before booking
* Footer with restaurant information
* Responsive layout using CSS
* React components for reusable UI
* Form handling using useState hook

## Technologies Used

* React
* JavaScript
* HTML5
* CSS3
* React Hooks (useState, useEffect)
* Basic API simulation

## Booking Functionality

The booking page allows the user to:

* Select date
* Select time
* Select number of people
* Submit the form

The form is controlled using React state.

## How to Run the Project

1. Install dependencies

npm install

2. Start the project

npm start

3. Open in browser

http://localhost:3000
